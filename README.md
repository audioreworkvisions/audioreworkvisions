## 🚀 Was ich baue
- **OIOSAI / OIMM** – agentische KI-Desktop-App (Electron + React) mit **Python FastAPI**, **lokalen ONNX-Modellen** und **hybriden Azure-Endpoints**.  
- **Legal-Office-Hub** – schlanke RAG-Workflows (PDF-OCR, Chunking, Embeddings, Azure AI Search/FAISS).  
- **PolyMorph.AI – Arena of Minds** – modulare Tools für Bild/Audio/Video-Generierung (SDXL-ONNX, Whisper-tiny, Sora-Prompts).

> **Prinzip:** kleine, schnelle Modelle 🏎️ + robuste UX 🧭 + klare Automatisierung 🛠️.

## 🌟 Wie ich baue
- **Frontend:** Electron · React/Next.js · Tailwind · shadcn/ui · Zustand  
- **Backend:** Python · FastAPI · Uvicorn · WebSockets · SSE  
- **KI/ML:** ONNX Runtime GenAI · OpenVINO · DirectML · Phi-4/3 Mini · lokale Embeddings  
- **RAG:** Chroma/FAISS · Azure AI Search · PDF-OCR (Tesseract/Camelot)  
- **Infra/Build:** PNPM · Vite · Electron Builder · uv/Poetry · Docker (leichtgewichtig)  
- **Daten:** SQLite (persistente Chats) · Supabase Auth (minimal, sicher)
